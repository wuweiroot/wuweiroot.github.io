    内存四区#

    运行之前#

    代码区#

    存放 CPU 执行的机器指令。通常代码区是可共享的（即另外的执行程序可以调用它），使其可共享的目的是对于频繁被执行的程序，只需要在内存中有一份代码即可。代码区通常是只读的，使其只读的原因是防止程序意外地修改了它的指t令。另外，代码区还规划了局部变量的相关信息。

    全局初始化数据区/静态数据区（data段）#

    该区包含了在程序中明确被初始化的全局变量、已经初始化的静态变量（包括全局静态变量和t）和常量数据（如字符串常量）。

    未初始化数据区（又叫 bss 区）#

    存入的是全局未初始化变量和未初始化静态变量。未初始化数据区的数据在程序开始执行之前被内核初始化为 0 或者空（NULL）。

    运行之后#

    程序在加载到内存前，代码区和全局区(data和bss)的大小就是固定的，程序运行期间不能改变。然后，运行可执行程序，操作系统把物理硬盘程序load(加载)到内存，除了根据可执行程序的信息分出代码区（text）、数据区（data）和未初始化数据区（bss）之外，还额外增加了栈区、堆区。

    • 代码区（text segment） 加载的是可执行文件代码段，所有的可执行代码都加载到代码区，这块内存是不可以在运行期间修改的。

    • 未初始化数据区（BSS） 加载的是可执行文件BSS段，位置可以分开亦可以紧靠数据段，存储于数据段的数据（全局未初始化，静态未初始化数据）的生存周期为整个程序运行过程。  全局初始化数据区/静态数据区（data segment） 加载的是可执行文件数据段，存储于数据段（全局初始化，静态初始化数据，文字常量(只读)）的数据的生存周期为整个程序运行过程。

    • 栈区（stack） 栈是一种先进后出的内存结构，由编译器自动分配释放，存放函数的参数值、返回值、局部变量等。在程序运行过程中实时加载和释放，因此，局部变量的生存周期为申请到释放该段栈空间。

    • 堆区（heap） 堆是一个大容器，它的容量要远远大于栈，但没有栈那样先进后出的顺序。用于动态内存分配。堆在内存中位于BSS区和栈区之间。一般由程序员分配和释放，若程序员不释放，程序结束时由操作系统回收。

    C语言关键字#

    volatile#

    关键字的主要作用是告诉编译器不要对该变量进行优化，因为该变量的值可能在编译器无法预料的情况下发生变化。这种情况通常发生在硬件相关的编程中，比如与外部设备通信、中断服务程序等。在这些情况下，编译器可能会对变量的读取和写入进行优化，导致程序出现意料之外的行为。使用volatile关键字可以确保编译器不对变量进行优化，每次访问变量时都会从内存中读取最新的值。

    static关键字#

    有多种含义，主要取决于它的位置和用法：

    全局变量#

    在全局变量前使用static关键字，表示该变量的作用域仅限于当前文件，其他文件无法访问。这样可以避免全局变量在不同文件中引起命名冲突。

    局部变量#

    在局部变量量前使用static关键字，表示该变量的生命周期延长到整个程序运行期间，而不是函数执行期间。这样可以使得变量在函数调用结束后仍然保留其值，下次调用函数时可以继续使用。

    函数#

    在函数前使用static关键字，表示该函数的作用域仅限于当前文件，其他文件无法调用该函数。这样可以实现函数的封装和隐藏，避免被其他文件调用。

    const#

    const 关键字用于定义常量，表示变量的值在定义后不能被修改。一旦使用 const 关键字声明一个变量为常量，编译器将会确保在程序的执行过程中不会对该变量的值进行修改，否则会导致编译错误。常量在程序中起到固定数值的作用，可以提高程序的可读性和可维护性。

    定义常量：#

    通过 const 关键字定义一个常量，例如 const int MAX_VALUE = 100;。

    函数参数修饰：#

    在函数参数中使用 const 关键字修饰某个参数，表示该参数在函数内部不可被修改，可以增加函数的安全性。

    指针常量：#

    可以使用 const 关键字定义指针常量，表示指针指向的内存地址不可变，但指针本身可以改变。

    常量指针：#

    使用 const 关键字定义常量指针，表示指针本身不可变，但指针指向的内存地址可以改变。

    extern#

    声明全局变量：#

    在一个文件中使用 extern 关键字声明一个全局变量，表示该变量的实际定义在其他文件中。例如： extern int globalVar; // 在当前文件中声明全局变量 globalVar，实际定义在其他文件中

    声明全局函数：#

    在一个文件中使用 extern 关键字声明一个全局函数，表示该函数的实际定义在其他文件中。例如： extern void function(); // 在当前文件中声明全局函数 function，实际定义在其他文件中

    引用外部变量或函数：#

    在当前文件中使用 extern 关键字引用其他文件中定义的全局变量或函数。例如： extern int globalVar; // 引用其他文件中定义的全局变量 globalVar extern void function(); // 引用其他文件中定义的全局函数 function
